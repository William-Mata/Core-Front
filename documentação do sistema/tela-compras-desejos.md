# Compras - Lista de Desejos

## O que voce faz nesta tela
- Cadastra itens que deseja comprar futuramente.
- Informa quantidade e valor alvo.
- Seleciona varios desejos e converte para uma nova lista de compras.

## Regras importantes
- Para converter, e obrigatorio selecionar pelo menos um desejo.
- A conversao pede confirmacao antes de executar.
